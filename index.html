    About The Page

    ## Description

    This is a web application that generates random jokes and allows users to add comments. It utilizes the Joke API to fetch jokes and displays them on the page. Users can also leave comments, which are stored locally in the browser's storage.


    ## Features

    - Fetches random jokes from the Joke API - Displays jokes and random images on the page - Allows users to add comments - Comments are stored in the browser's local storage (only read and write in user browser and comments are not share with others.) - Provides a countdown timer for the next joke update
